Usage:
    md-secretary compress [options] <project-name>

Options:
    --path, -p <path-to-notebook>
        Specifies the path to a notebook. By default, the current working directory is used.

    --output, -o <path-to-output-location>
        Specfies the output path to the generated JSON file. Default is the current working directory.

    --transfer, -t
        Removes the project from the notebook after generating the JSON file.

    --help, -h
        Displays this help message and exits.
