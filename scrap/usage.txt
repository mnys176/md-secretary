Usage:
    md-secretary scrap [options] <project-name>

Options:
    --path, -p <path-to-notebook>
        Specifies the path to a notebook. By default, the current working directory is used.

    --force, -f
        Disables any confirmation and performs the operation automatically. Use this flag with caution.

    --help, -h
        Displays this help message and exits.
